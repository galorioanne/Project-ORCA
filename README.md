# Project CARO
> Coordinate. Align. Resolve. Organize.
>
### Project Description
A simple, flexible program that helps organize tasks and create schedules. Make planning your week (or maybe more than a week...) easier!
### Features
- Organized and separate pages for tasks and schedules
- Write down tasks 
- Write down activties in your schedule 
- As project CARO is not an online space, progress is automatically saved even after closing the program.
### How To Run
- Download ZIP file
- Run caro.exe
- That's it! =)

# Project CARO
> Coordinate. Align. Resolve. Organize.
### Project Description
A flexible program that helps organize tasks and create schedules that do not conflict eachother. Make 
communication and planning your week easier!
### Features
- Organized and separate pages for tasks and schedules
- Write down tasks 
- Make your schedule
### How To Run
- Download the repository (Do not download caro.exe exclusively, all require each other to operate)
- Run caro.exe
- CTRL+CLICK `http://127.0.0.1:5000/` on the console or copy-paste to browser of choice
### How To Exit
- Close browser
- Return to console/Command Prompt
- Click CTRL+C
